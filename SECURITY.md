# Security policy

Please report vulnerabilities privately through GitHub's private vulnerability
reporting for this repository. Do not include real birth data in a report.

Supported releases are the latest tagged minor release and the current default
branch. Reports should include impact, affected versions, reproduction steps,
and a minimal synthetic proof.

We will acknowledge valid reports when possible, coordinate a fix, and credit
reporters who want public recognition. This project does not offer a bug bounty
or guaranteed response time.
